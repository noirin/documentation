Implementation of com.xmlmind.ditac.preprocess.PreProcessor, the ditac preprocessor.

The PreProcessor converts a DITA map file (or a list of topic files, in which case a DITA map is generated on the fly) to one or more .ditac files and a single ditac_lists.ditac_lists file.

The PreProcessor has been designed in order to be easily embedded in any Java desktop or server-side application. However an even easier alternative to embedding PreProcessor consists in embedding com.xmlmind.ditac.convert.Converter.
